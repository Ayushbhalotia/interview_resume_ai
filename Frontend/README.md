# GenAI Interview Preparation App

A full-stack AI application that helps users prepare for interviews using Generative AI.

## Features
- Resume upload and analysis
- AI-generated interview questions
- Skill gap detection
- ATS resume suggestions

## Tech Stack
- React (Frontend)
- Node.js + Express (Backend)
- Gemini API (AI)
- JWT Authentication

## How it works
User uploads resume → Backend processes → AI generates insights → Displayed in UI
